A Streamlit web app that allows you to summarize content from **YouTube videos** or **websites** using **LangChain** and **Hugging Face models**.

---

## Features

- Input a **YouTube URL** or **website URL**.
- Summarize the content into ~300 words.
- Uses **Hugging Face models** for natural language summarization.
- Built with **Streamlit** for an interactive web interface.
